# Hotel Occupancy Report

A C++ application that calculates hotel occupancy rates and generates a detailed occupancy report for multiple floors.

## Features

- Validates user input
- Tracks room occupancy by floor
- Calculates floor occupancy percentages
- Calculates overall hotel occupancy rate
- Generates formatted reports using file output

## Technologies

- C++
- Loops
- Conditional Statements
- Input Validation
- File I/O
- Data Processing
